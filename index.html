hey! my name is Claire Durant. I'm a software engineer with a background in operating systems development. in addition to my core engineering work, I place a lot of emphasis on creating approachable technical content for readers of varying backgrounds.

since 2018, I've been working with IBM in the z/TPF development lab. z/TPF is an operating system for IBM Z mainframes built with an emphasis on high-volume transaction processing for business-critical operations.

publications & patents

'Centralized database system with geographically partitioned data'

this patent details a system for storing data in a number of remote data stores. this system temporarily stores data on the main host system while waiting for an asynchronous process to update the data on the remote data store. through the use of this asynchronous process, users of this system can update a piece of data and continue processing without waiting for network communications with the remote data store to complete.
